# Report API
## Тестовое задание для позиции стажёра backend-разработчика

# Описание
Реализован эндоинт
```http
POST /public/report/export
```
Принимает Текстовый файл и возвращает excel-файл (.xlsx) с частотной статистикой словоформ

---
## Функциональность
- Обработка больших файлов (до гигабайтов) без загрузки в память
- Приведение слов к нормальной форме (лемматизация)
- Подсчёт:
  - Общего количества словоформ в файле
  - количество по строкам
- Генерация Excel-файл
---
## Архитектура
```mermaid
api/         # HTTP слой (роуты)
services/    # Бизнес логика
main.py      # Точка входа
```

## Технологии
- FastAPI   # Веб фреймворк
- Uvicorn   # сервер
- Stanza    # лемматизация
- openpyxl  # генерация .xlsx файл
---

## Установка и запуск
1. Клон пректа
```bash
git clone <repo url>
cd <project folder>
```
2. Установить зависимости
```bash 
pip install < requirements.txt
```
3. Скачать модель
При первом запуске в файле main.py раскомментировать строку

```python
import stanza
stanza.download("ru")
```

4. запуск 
```bash
uvicorn main:app --reload
```

5. Тест
через Swagger
http://127.0.0.1:8000/docs
